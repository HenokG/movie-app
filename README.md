# Movie app client with microservices architecture

## Technologies
  - React
  - Typescript
  - Graphql
  - Docker
  - Docker Compose
  - Kubernetes
  - AWS
  - GithubActions
  - Husky
  - Commitlint
  - React-Testing-Library
  - Jest
  
## How to start
  
  simple, just run `docker compose up`, it will launch the container with mounted volumes, head up to http://localhost:3000 to view app
